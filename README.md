# Front-ledgerly (Frontend móvil)

> Aplicación móvil frontend de **Ledgerly** — interfaz para gestionar registros y transacciones financieras.  
> Proyecto basado en **React Native con Expo** para desarrollo, pruebas y ejecución.

---


## Descripción

**Front-ledgerly** es la aplicación móvil de Ledgerly, encargada de gestionar cuentas, transacciones y reportes financieros.  
La app se ejecuta mediante **Expo** y consume una API backend para almacenamiento, autenticación y lógica financiera.

---

## Características principales

- Autenticación (login / logout, persistencia de token)
- Dashboard con métricas financieras
- CRUD de cuentas y transacciones
- Listados con búsqueda y filtros
- Formularios con validación y manejo de errores
- Diseño responsivo para múltiples resoluciones
- Integración con backend vía API

---

## Tecnologías

- **React Native**
- **Expo**
- TypeScript

---

## Requisitos

- Node.js `>= 16`
- npm o yarn
